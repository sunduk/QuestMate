# QuestMate
개인의 성취(Self-Improvement)와 타인에 대한 기여(Altruism)를 동시에 충족시키는 단기 루프형 동기부여 플랫폼

---

# [Project: Quest Mate] 기획 요약서

### 1. 서비스 개요 (Concept)
*   **슬로건:** "함께하면 습관이 되고, 성공하면 내 방이 된다."
*   **장르:** 소셜 챌린지 + 하우징(Housing/꾸미기) 시뮬레이션
*   **핵심 가치:** 개인의 성취(Self-Improvement)와 타인에 대한 기여(Altruism)를 동시에 충족시키는 **단기 루프형 동기부여 플랫폼**.
*   **타겟 유저:** 작심삼일이 일상인 MZ/잘파 세대, 힐링과 소셜 활동을 동시에 원하는 유저.

### 2. 핵심 루프 (Core Loop)
1.  **파티 결성 (Match):** 1일에서 1주일 단위의 단기 퀘스트 파티 생성 또는 참가 (최소 2인, 최대 6인)
2.  **퀘스트 수행 (Action):** 정해진 시간에 미션 수행 후 사진 인증.
3.  **상호 검증 & 협력 (Interaction):**
    *   파티장/파티원 간 상호 인증 (좋아요/승인).
    *   실패 위기 동료 구제(부활) 및 응원.
4.  **보상 획득 (Reward):** 퀘스트 완료 시 '골드/포인트' 획득 (파티 전원 성공 시 보너스 배율 적용).
5.  **하우징 (Deco):** 획득한 포인트로 가구/소품 구매 및 '내 방' 꾸미기.
6.  **순환 (Repeat):** 파티 해산 후, 새로운 테마의 퀘스트를 찾아 재도전.

### 3. 차별화 포인트 (USP)
**A. 게임 개발자의 '손맛'이 담긴 하우징 시스템**
*   단순 이미지 나열이 아닌, **Unity 타일맵 방식**을 웹(Canvas/Pixi.js)에 구현.
*   가구 배치, 회전, 레이어 우선순위 조절 등 정교한 조작감 제공.
*   귀여운 2D 도트/벡터 스타일의 비주얼 보상 제공.

**B. '이타심'을 자극하는 소셜 시스템 (Co-op)**
*   **부활(Revive):** 나의 여유 자원으로 실패한 파티원을 구제 (영웅 심리).
*   **파티 버프:** 파티원이 인증할 때마다 전체 보상 배율 증가 (공동 운명체).
*   **선물하기:** 특정 퀘스트에서만 얻는 '희귀 가구'를 파티원에게 선물 (유대감 강화).

**C. 부담 없는 단기 세션**
*   지루한 장기 프로젝트가 아닌, **세션 기반(Session-based)** 진행.
*   매주 새로운 사람들과 새로운 목표로 시작하여 리텐션 유지.

### 4. 기술 스택 (Tech Stack) - 게임 개발자 친화적 구성
*   **Client:** React (Next.js) + **HTML5 Canvas (Pixi.js or Konva.js)** for Housing View.
    *   *상태 관리: Zustand / 스타일: Tailwind CSS*
*   **Server:** **ASP.NET Core (C#)** - 고성능/안정성 확보.
*   **Database:** PostgreSQL (Entity Framework Core) or MSSQL.
*   **Infra:** Vercel (Front) + Azure/AWS or Supabase (Back/DB).
*   **AI (Future):** OpenAI Vision API를 활용한 인증샷 1차 필터링 및 응원 봇.

### 5. 개발 로드맵 (MVP)
*   **Phase 1 (기반 구축):** 회원가입, 퀘스트 CRUD, 기본 파티 시스템(승인/반려), 포인트 로직.
*   **Phase 2 (하우징):** 상점, 인벤토리, Canvas 기반 방 꾸미기 에디터 구현.
*   **Phase 3 (소셜 강화):** 부활 시스템, 파티 버프, 실시간 알림 구현.

**[기대 효과]**
웹 서비스의 접근성에 게임의 문법(보상, 협동, 꾸미기)을 완벽하게 이식하여, 기존 챌린지 앱들이 해결하지 못한 **'지속적인 재미'** 문제를 해결함.
